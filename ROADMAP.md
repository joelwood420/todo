ROADMAP: Full-stack TODO Web App (Flask + SQLite + Jinja2)

Summary
- Goal: Minimal, secure, maintainable TODO app with user auth, per-user todos, CRUD, tests, and Render deployment.

Phases (one-line each)
1. Setup: scaffold app, venv, requirements, app factory
2. Models/DB: SQLAlchemy User and Todo models, DB init
3. Routes/Templates: todos CRUD, Jinja2 templates, layout
4. Auth: registration, login/logout, session management
5. Tests: unit and integration tests, pytest
6. Deploy: add Procfile, env vars, Render docs

Progress: Setup, Models/DB, Routes/Templates completed. Tests in progress; Auth and Deploy pending.
